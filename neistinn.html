Instructions

Neistinn (in English ‘the spark’) trains learners of Icelandic to build simple sentences composed of subject, main verb, object and information about time or place. The app generates a sequence of words in their default form (e.g. nominative singular for nouns, infinitive for verbs, etc.) which need to be inflected.

To start, click on the button Generate a sentence. A randomized subject, main verb and object will appear. The user can also ask the generator to add an adjective to the subject or the object when the button +adj appears. At that point, the user can type in the text boxes the correct Icelandic form of the subject, the verb and the object and click on Verify to check whether the answer given is correct.

The items correctly translated will be marked in green and the incorrect ones in red. If no answer is given, the items will be marked in blue. The examples previously generated will be displayed on the side panel. The previous examples where the answer was incorrect are marked in red.

The various components of the sentences are marked with different colors and indications are given e.g. on which tense needs to be used, or whether there is an article attached to the noun, or whether the noun must be in the plural etc. A comprehensive list is shown here:

Colors

hundur This box type is used for nouns and pronouns
gamall This box type is used for adjectives
fara This box type is used for verbs
til This box type is used for prepositions and particles
minn This box type is used for possessive pronouns
←nf. This box type is used to signal the correct case after the answer has been given

Abbreviations

(FT.) = plural (fleirtala)
+GR. = with article (greinir)
(NT.) = present tense (nútíð)
(ÞT.) = past tense (þátíð)
(S.NT.) = present progressive (samsett nútíð), e.g. Ég er að...
(S.ÞT.) = past progressive (samsett þátíð), e.g. Ég var að...
NF. = nominative (nefnifall)
ÞF. = accusative (þolfall)
ÞGF. = dative (þágufall)
EF. = genitive (eignarfall)
